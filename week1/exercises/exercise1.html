Exercise 1: Your First HTML Page

Instructions

In this exercise, you'll create your very first HTML page from scratch. This will help you understand the basic structure of an HTML document.

Learning Objectives:

  • Create a complete HTML document with proper structure
  • Understand the purpose of different HTML elements
  • Practice writing clean, well-formatted HTML code

Tasks:

  1. Create a new file named first-page.html
  2. Add the DOCTYPE declaration
  3. Create the HTML root element with appropriate language attribute
  4. Add head and body sections
  5. In the head section, add:
    • Character encoding meta tag
    • Viewport meta tag
    • Page title: "My First HTML Page"
  6. In the body section, add:
    • A main heading (h1) with your name
    • A subheading (h2) that says "My First HTML Page"
    • A paragraph about why you're learning HTML
    • Another paragraph about what you hope to create with HTML
  7. Add at least one HTML comment in your code

Expected Output:

Your page should display:

  • Your name as the main heading
  • "My First HTML Page" as a subheading
  • Two paragraphs of text about your HTML journey

When you view the page source, it should have proper structure with all required elements.

Validation:

After completing your page, validate it using the W3C Markup Validation Service. Fix any errors before submitting.
