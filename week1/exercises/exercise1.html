Exercise 1: Your First HTML Page

Task:

Create a new HTML file called "first-page.html" in the exercises folder and build a basic HTML page that includes the following:

  1. Proper HTML5 document structure
  2. A title in the head section: "My First HTML Page"
  3. A main heading (h1) with the text: "Hello, World!"
  4. A paragraph that introduces yourself
  5. A second-level heading (h2) with the text: "About HTML"
  6. A paragraph explaining what HTML is

Hint:

Remember to use the basic HTML structure: DOCTYPE declaration, html, head, and body elements. Save your file with the .html extension and open it in a web browser to see the result.

Expected Result

Your page should display a heading "Hello, World!" followed by your introduction paragraph, then an "About HTML" heading with an explanation paragraph.

Solution Template

If you're stuck, here's a template to start with:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title></title>
</head>
<body>
    <!-- Add your h1 heading here -->
    
    <!-- Add your introduction paragraph here -->
    
    <!-- Add your h2 heading here -->
    
    <!-- Add your HTML explanation paragraph here -->
</body>
</html>

Check Your Work

After completing the exercise:

  1. Open your HTML file in a web browser
  2. Check that all elements appear as expected
  3. Validate your HTML using the W3C Markup Validation Service
