Week 1: HTML Fundamentals

Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It describes the structure of a web page semantically using a series of elements.

What is HTML?

HTML consists of a series of elements that you use to enclose, or wrap, different parts of the content to make it appear a certain way or act a certain way. The enclosing tags can make a word or image hyperlink to somewhere else, can italicize words, can make the font bigger or smaller, and so on.

HTML Versus Other Technologies

  • HTML - Structure of the content
  • CSS - Presentation and styling
  • JavaScript - Behavior and interactivity

HTML documents have the file extension .html or .htm

Basic Document Structure

Every HTML document follows a basic structure:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document Title</title>
</head>
<body>
    <!-- Content goes here -->
</body>
</html>

Element Breakdown

  • <!DOCTYPE html> - Declaration that tells the browser this is an HTML5 document
  • <html> - The root element that contains all content
  • <head> - Contains meta-information about the document
  • <meta> - Provides metadata such as character encoding and viewport settings
  • <title> - Sets the title of the page (appears in browser tab)
  • <body> - Contains all the visible content of the page

The lang attribute in the HTML tag helps screen readers and search engines understand the language of your content.

Text Elements

Headings

HTML provides six levels of headings, from <h1> (most important) to <h6> (least important). 

<h1>This is a Level 1 Heading</h1>
<h2>This is a Level 2 Heading</h2>
<h3>This is a Level 3 Heading</h3>
<h4>This is a Level 4 Heading</h4>
<h5>This is a Level 5 Heading</h5>
<h6>This is a Level 6 Heading</h6>

Paragraphs

The <p> element is used to create paragraphs of text. 

<p>This is a paragraph of text. HTML will automatically wrap the text and create a new line when needed.</p>
<p>This is a second paragraph. Notice how paragraphs are separated with a margin.</p>

Line Breaks

To insert a line break without creating a new paragraph, use the <br> element. 

<p>This is a paragraph with a line break<br>right in the middle.</p>

Horizontal Rule

The <hr> element creates a horizontal line, useful for thematic breaks. 

<p>This is above the line.</p>
<hr>
<p>This is below the line.</p>

Formatting Text

HTML includes elements for formatting text:

Bold and Italic

<p>This is <strong>bold text</strong> and this is <em>italic text</em>.</p>

Subscript and Superscript

<p>This is a formula: H<sub>2</sub>O and this is 2<sup>3</sup> equals 8.</p>

Other Formatting Elements

  • <mark> - Highlighted text
  • <del> - Deleted text
  • <ins> - Inserted text
  • <small> - Smaller text
  • <pre> - Preformatted text (preserves spaces and line breaks) 
<p>This is <mark>highlighted</mark> text.</p>
<p>This is <del>deleted</del> and <ins>inserted</ins> text.</p>
<p>This contains <small>smaller text</small> within a paragraph.</p>

Semantic elements like <strong> and <em> are preferred over <b> and <i> as they convey meaning, not just appearance.

HTML Comments

HTML comments are not displayed in the browser but can be viewed in the source code. They're useful for documenting your code. 

<!-- This is a comment -->

<!-- 
    This is a 
    multi-line 
    comment 
-->

Common Uses for Comments

  • Explaining complex code sections
  • Temporarily disabling code
  • Marking the beginning and end of content sections
  • Adding reminders or todos
