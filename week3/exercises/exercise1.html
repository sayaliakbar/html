Exercise 1: Creating Basic Forms

Task:

Create a new HTML file called "contact-form.html" in the exercises folder and implement a basic contact form with the following elements:

  1. A form with appropriate action and method attributes
  2. Text inputs for name, email, and subject
  3. A textarea for the message
  4. A dropdown menu for the contact reason (options: Question, Feedback, Support, Other)
  5. A submit button
  6. A reset button

Hint:

Remember to use appropriate name attributes for all form controls. Use labels for each form element to improve accessibility, and organize the form logically.

Expected Result

Your form should include all the required elements, be properly structured, and have a logical tab order. All form controls should be properly labeled.

Check Your Work

After completing the exercise:

  1. Open your HTML file in a web browser
  2. Test that all form elements appear correctly
  3. Check that clicking on labels focuses the corresponding input
  4. Test the tab order moves logically through the form
  5. Try the reset button to ensure it clears the form
  6. Validate your HTML using the W3C Markup Validation Service
