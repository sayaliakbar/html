Exercise 1: Identifying Semantic Elements

Task:

Create a new HTML file called "semantic-conversion.html" in the exercises folder. In this exercise, you'll practice converting non-semantic HTML to semantic HTML.

  1. Copy the non-semantic HTML code provided below into your file
  2. Convert the structure to use appropriate semantic HTML5 elements
  3. Replace all <div> elements with appropriate semantic elements such as <header>, <nav>, <main>, <article>, <section>, <aside>, and <footer>
  4. Add comments explaining your choices for each major section

Hint:

Think about the meaning and purpose of each section when choosing the appropriate semantic element. Consider what the content represents, not how it looks.

Expected Result

Your semantic HTML version should:

  • Use appropriate semantic elements for each content section
  • Include comments explaining your element choices
  • Maintain the same content and structure
  • Be properly indented and formatted

Check Your Work

After completing the exercise:

  1. Open your HTML file in a web browser to ensure it displays correctly
  2. Check that you've replaced all non-semantic <div> elements
  3. Ensure proper nesting of semantic elements
  4. Validate your HTML using the W3C Markup Validation Service

Submission Guidelines

When submitting your work, be prepared to:

  1. Explain your choices for each semantic element
  2. Discuss how the semantic structure improves the page's accessibility and SEO
  3. Share any challenges you faced during the conversion process
