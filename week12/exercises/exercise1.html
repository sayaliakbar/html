Exercise 1: PWA Evaluation and Analysis

Objective

Learn to identify and evaluate Progressive Web Applications in the wild, understanding their core features and implementation strategies.

Instructions

  1. Visit the following PWAs and evaluate them based on the criteria below:
    • Starbucks
    • Twitter
    • Spotify
    • A PWA of your choice
  2. For each application, complete the evaluation table below.
  3. Use Chrome DevTools (Application tab) to analyze the implementation details.
  4. Write a brief report comparing these PWAs and what you've learned from analyzing them.

Evaluation Table Template

Feature Implementation Details Score (1-5) Notes

Performance
User Experience

Report Guidelines

In your report (approximately 500 words), address the following:

  • Which PWA performed best overall and why?
  • Common patterns you noticed across different PWAs
  • Innovative features or approaches that stood out
  • Areas where the PWAs could improve
  • What lessons you can apply to your own PWA project

Submission

Submit your completed evaluation tables and report in a PDF or HTML document.
