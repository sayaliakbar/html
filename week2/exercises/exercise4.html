Exercise 4: Tables

Task 1: Basic Table

Create a simple 3x3 table showing data for three students with columns for Name, Age, and Grade.

Hint: Use tr for rows and td for data cells. Use th for header cells.

Task 2: Table with Headers

Create a table of products with columns for Product Name, Price, and Stock. Include appropriate header cells.

Hint: Use th elements for the header row.

Task 3: Semantic Table Structure

Create a table showing quarterly sales data for two years with appropriate caption, header, body, and footer sections.

Hint: Use caption, thead, tbody, and tfoot elements.

Task 4: Cell Spanning

Create a class schedule table with time slots as rows and days of the week as columns. Some classes should span multiple time slots (using rowspan) and some events should span multiple days (using colspan).

Hint: Use rowspan and colspan attributes to make cells span multiple rows or columns.

Task 5: Column Groups

Create a financial report table with column groups to style different sections differently.

Hint: Use colgroup and col elements to group and style columns.
