Exercise 3: Lists

Task 1: Unordered List

Create an unordered list of your five favorite foods.

Hint: Use the ul element with li elements inside.

Task 2: Ordered List

Create an ordered list of the top 5 tallest buildings in the world.

Hint: Use the ol element with li elements inside.

Task 3: Definition List

Create a definition list that defines at least three HTML terms (like HTML, CSS, DOM, etc.).

Hint: Use dl with dt (term) and dd (definition) elements.

Task 4: Nested Lists

Create a nested list representing a categorized shopping list:

  • Produce (with nested items: apples, bananas, carrots)
  • Dairy (with nested items: milk, cheese, yogurt)
  • Bakery (with nested items: bread, muffins, bagels)

Hint: Place a ul or ol element inside an li element to create a nested list.

Task 5: Custom List Attributes

Create an ordered list of the days of the week starting with "Monday," but make sure the numbering starts at 1 (not 0).

Create another ordered list of three items using Roman numerals.

Create an ordered list that counts backwards from 5 to 1.

Hint: Use the start, type, and reversed attributes.
