Exercise 1: Links and Anchors

Task 1: Absolute Links

Create links to three different external websites. Make at least one of them open in a new tab.

Hint: Use target="_blank" to open a link in a new tab.

Task 2: Relative Links

Create a file called "about.html" in this same folder, then create a link to it below.

Now, imagine you have a file in a subfolder called "products/catalog.html" and create a link to it.

Hint: For subfolder links, include the folder path in your href.

Task 3: Email and Phone Links

Create a link that opens the user's email client to send an email to "contact@example.com" with the subject "Website Inquiry".

Create a link that allows mobile users to call the number +1234567890.

Hint: Use mailto: for email links and tel: for phone links.

Task 4: Page Anchors

Create a table of contents at the top of the page with links to the following sections:

  • Introduction
  • Chapter 1
  • Chapter 2
  • Summary

Hint: Create sections with IDs below, then link to them from here.

Introduction

This is the introduction section of our document.

Chapter 1

This is the content for chapter 1.

Chapter 2

This is the content for chapter 2.

Summary

This is the summary of our document.

Task 5: Advanced Link Attributes

Create a link to download a file called "report.pdf". Add a title attribute that shows a tooltip when hovering over the link.

Create a link to a partner website (use any URL) that uses the rel="nofollow" attribute.

Hint: The download attribute tells browsers to download the resource instead of navigating to it.
