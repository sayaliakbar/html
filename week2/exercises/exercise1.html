Exercise 1: Creating Different Types of Links

Task:

Create a new HTML file called "links.html" in the exercises folder and implement the following types of links:

  1. An absolute link to a website you find useful for HTML learning
  2. A relative link to the Week 1 lesson.html file
  3. An email link that opens an email to a fictional address: "contact@example.com"
  4. A phone link to a fictional phone number
  5. Create at least 3 sections with IDs in your document and add bookmark links to navigate between them
  6. Add a link that opens in a new tab with the proper security attributes

Hint:

Remember to use the appropriate attributes for each link type. For links that open in a new tab, include target="_blank" rel="noopener noreferrer" for security.

Expected Result

Your page should include various types of working links that demonstrate the different use cases for anchors in HTML.
