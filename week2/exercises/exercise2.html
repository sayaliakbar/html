Exercise 2: Images and Image Maps

Task 1: Basic Images

Add an image to this page with appropriate alt text. You can use the placeholder image URL: https://via.placeholder.com/300x200

Hint: Don't forget the alt attribute for accessibility.

Task 2: Image Dimensions

Add another image and specify its width and height attributes. Use the placeholder: https://via.placeholder.com/600x400

Hint: Setting width and height helps prevent layout shifts when the page loads.

Task 3: Responsive Images

Create a responsive image that uses the srcset attribute to provide different image sizes based on the viewport. For this exercise, you can use these placeholder URLs:

  • Small: https://via.placeholder.com/300x200
  • Medium: https://via.placeholder.com/600x400
  • Large: https://via.placeholder.com/900x600

Hint: Use srcset with width descriptors (300w, 600w, 900w).

Task 4: Figure and Caption

Use the figure and figcaption elements to add a captioned image.

Hint: The figure element represents self-contained content, and figcaption provides a caption.

Task 5: Image Map

Create an image map using this placeholder image: https://via.placeholder.com/600x400

Define at least three clickable areas on the image, each linking to a different webpage.

Hint: Use the map and area elements with appropriate shape and coords attributes.
