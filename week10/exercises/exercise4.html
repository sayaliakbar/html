Tips:

  • Remember the different attribute selectors and their meanings:
    • [attr] - Element has the attribute
    • [attr="value"] - Attribute equals value exactly
    • [attr^="value"] - Attribute starts with value
    • [attr$="value"] - Attribute ends with value
    • [attr*="value"] - Attribute contains value anywhere
    • [attr~="value"] - Attribute contains value as a space-separated word
    • [attr|="value"] - Attribute equals value or starts with "value-"
  • You can combine multiple attribute selectors: input[type="text"][required]
  • For case-insensitive matching, add "i" after the closing bracket: [attr="value" i]
  • Data attributes follow the same rules: [data-category="electronics"]
  • You can combine attribute selectors with other selectors: a[href^="http"]

When you've completed the exercises, test your CSS selectors by applying them and observing the results.
