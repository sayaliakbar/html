Exercise 2: Creating Social Media Cards

Instructions

In this exercise, you'll implement Open Graph and Twitter Card metadata to optimize how your webpage appears when shared on social media platforms.

You'll work with a product page for a fictional online store and add the appropriate metadata to create engaging social media previews.

Tasks

  1. Create a new HTML file named "social-cards.html"
  2. Add the basic HTML structure with standard metadata (charset, viewport, etc.)
  3. Implement Open Graph metadata for Facebook and other platforms:
    • og:title - Create a compelling title for social media
    • og:description - Write an engaging description (under 200 characters)
    • og:image - Specify an image URL (assume 1200Ã—630px dimensions)
    • og:url - Set the canonical URL for the product
    • og:type - Set the appropriate content type
    • og:site_name - Add your store's name
  4. Implement Twitter Card metadata:
    • twitter:card - Choose the appropriate card type
    • twitter:title - Create a title for Twitter
    • twitter:description - Write a concise description
    • twitter:image - Specify the same or a different image
    • twitter:site - Add a Twitter username
  5. Add basic page content to showcase the product

Example Solution

Here's a partial solution to help you get started with implementing social media cards: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ultra Noise-Canceling Headphones | EcoTech Audio</title>
    
    <!-- Open Graph Metadata for Facebook, LinkedIn, etc. -->
    <meta property="og:title" content="Eco-Friendly Premium Noise-Canceling Headphones" />
    <meta property="og:description" content="Experience 40 hours of immersive, crystal-clear audio with our sustainable headphones made from eco-friendly materials. Industry-leading noise cancellation." />
    <meta property="og:image" content="https://example.com/images/headphones-social.jpg" />
    <meta property="og:url" content="https://example.com/products/noise-canceling-headphones" />
    <meta property="og:type" content="product" />
    <meta property="og:site_name" content="EcoTech Audio" />
    
    <!-- Twitter Card Metadata -->
    <meta name="twitter:card" content="summary_large_image" />
    <meta name="twitter:site" content="@ecotechaudio" />
    <!-- Add the remaining Twitter Card metadata here -->
