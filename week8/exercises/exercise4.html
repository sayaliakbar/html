Exercise 4: Embedding Third-Party Widgets

Instructions

In this exercise, you'll practice embedding various third-party widgets and APIs into a web page to enhance its functionality and content.

You'll create a "Local Business Page" for a fictional café that includes multiple embedded widgets to provide visitors with useful information.

Tasks

  1. Create a new HTML file named "api-integration.html"
  2. Add the basic HTML structure with appropriate metadata
  3. Implement the following embedded content:
    • A Google Map showing the café's location
    • A weather widget showing current weather in the café's location
    • A social media feed (Twitter, Instagram, or Facebook)
    • A YouTube video showcasing the café or a related topic
    • A contact form or reservation widget
  4. Add appropriate error handling or fallback content for each widget
  5. Ensure all embedded content is responsive
  6. Add a privacy notice about third-party content

Note: For this exercise, you can use placeholder API keys or embed codes. In a real project, you would need actual API keys from each service.

Example Solution

Here's a partial solution showing how to implement some of the embedded widgets:

Google Maps Embed

<section id="location">
    <h2>Find Us</h2>
    <p>123 Main Street, Downtown, City, State 12345</p>
    
    <div class="embed-container">
        <iframe 
            class="responsive-iframe"
            src="https://www.google.com/maps/embed/v1/place?key=YOUR_API_KEY&q=123+Main+St,Downtown,City,State" 
            allowfullscreen
            loading="lazy"
            title="Urban Grove Café Location"
            onerror="this.style.display='none';this.nextElementSibling.style.display='block';"
        ></iframe>
        <div class="embed-fallback" style="display:none;">
            <p>Google Maps could not be loaded.</p>
            <p>Our café is located at 123 Main Street, Downtown.</p>
        </div>
    </div>
</section>

Weather Widget

<section id="weather">
    <h2>Current Weather at Our Location</h2>
    
    <div class="embed-container">
        <div id="weather-widget" class="responsive-iframe"></div>
        <div class="embed-fallback" id="weather-fallback" style="display:none;">
            <p>Weather information could not be loaded.</p>
            <p>Call us at (555) 123-4567 for current patio availability.</p>
        </div>
    </div>
</section>

<script>
    // Example weather widget implementation
    function loadWeatherWidget() {
        try {
            fetch('https://api.openweathermap.org/data/2.5/weather?q=YourCity&appid=YOUR_API_KEY')
                .then(response => {
                    if (!response.ok) throw new Error('Weather data not available');
                    return response.json();
                })
                .then(data => {
                    document.getElementById('weather-widget').innerHTML = `
                        <div style="padding: 20px; text-align: center;">
                            <h3>Weather in ${data.name}</h3>
                            <p>Temperature: ${Math.round(data.main.temp - 273.15)}°C</p>
                            <p>Condition: ${data.weather[0].description}</p>
                            <p>Humidity: ${data.main.humidity}%</p>
                        </div>
                    `;
                })
                .catch(error => {
                    document.getElementById('weather-widget').style.display = 'none';
                    document.getElementById('weather-fallback').style.display = 'block';
                    console.error('Error loading weather:', error);
                });
        } catch (error) {
            document.getElementById('weather-widget').style.display = 'none';
            document.getElementById('weather-fallback').style.display = 'block';
            console.error('Error in weather widget:', error);
        }
    }

    // Call the function when the page loads
    window.addEventListener('load', loadWeatherWidget);
</script>
