Exercise 1: Implementing Essential Metadata

Instructions

In this exercise, you'll practice implementing essential metadata tags to improve SEO and browser functionality for a website.

Use the template HTML file provided below and follow the tasks to add appropriate metadata.

Tasks

  1. Create a new HTML file named "metadata-practice.html"
  2. Add the basic HTML structure
  3. Implement the following metadata in the head section:
    • Character set declaration
    • Viewport settings for responsive design
    • A descriptive and SEO-friendly title (under 60 characters)
    • A meta description (under 160 characters) that summarizes the page content
    • Author information
    • Keywords relevant to the content (5-7 keywords)
  4. Add favicon information (you can use a placeholder URL)
  5. Add a canonical URL
  6. Add appropriate robots meta tags
