Exercise 3: Adding Structured Data

Instructions

In this exercise, you'll practice implementing structured data using both Microdata and JSON-LD formats to enhance how search engines understand and display your content.

You'll work with a recipe page and add appropriate structured data to make it eligible for rich results in search engines.

Tasks

  1. Create a new HTML file named "structured-data.html"
  2. Add the basic HTML structure with standard metadata
  3. Implement structured data for a recipe using both methods:
    • First, implement using Microdata directly in your HTML
    • Then, add the equivalent JSON-LD version in a script tag
  4. Include the following properties in your structured data:
    • Recipe name
    • Author
    • Image
    • Description
    • Preparation time and cooking time
    • Recipe yield
    • Ingredients (at least 5)
    • Instructions (at least 3 steps)
    • Nutrition information
  5. Style your recipe page to look like a typical recipe blog post
