Exercise 1: Working with Block and Inline Elements

Task:

Create a new HTML file called "elements.html" in the exercises folder. In this exercise, you'll practice using different block and inline elements to understand how they behave in the document flow.

  1. Create a document with proper HTML structure
  2. Add a main heading titled "Block and Inline Elements"
  3. Create a section for "Block Elements" with:
    • A level 2 heading
    • At least 3 different block elements (paragraphs, headings, lists, etc.)
  4. Create a section for "Inline Elements" with:
    • A level 2 heading
    • A paragraph that includes at least 4 different inline elements (span, strong, em, a, etc.)
  5. Create a section that demonstrates nesting elements:
    • Block elements containing inline elements
    • Block elements containing other block elements

Hint:

Remember that block elements create new lines and take up the full width available, while inline elements flow within the text and only take as much space as needed.

Expected Result

Your page should demonstrate the behavior of various block and inline elements, showing how they flow in the document and how they can be nested.

Solution Template

<body>
    <h1>Block and Inline Elements</h1>
    
    <section>
        <h2>Block Elements</h2>
        <!-- Add at least 3 different block elements here -->
    </section>
    
    <section>
        <h2>Inline Elements</h2>
        <p>
            <!-- Add a paragraph with at least 4 different inline elements -->
        </p>
    </section>
    
    <section>
        <h2>Nesting Elements</h2>
        <!-- Demonstrate nesting of block and inline elements -->
    </section>
</body>
</html>

Check Your Work

After completing the exercise:

  1. Open your HTML file in a web browser
  2. Observe how block elements create new lines
  3. Observe how inline elements flow within the text
  4. Check that your nesting demonstrates proper containment rules
